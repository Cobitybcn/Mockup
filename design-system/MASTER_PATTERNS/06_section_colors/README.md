# Master Pattern: Section Colors

## Pattern

Muted pigmentary pastel color identifies sections, workflow families, boards, and meaningful states. Color organizes the studio. It never decorates and never dominates.

## Invariants

- Keep the base environment warm, light, and neutral.
- Use dusty rose, sage, warm ochre, copper, soft yellow, muted lilac, muted blue, and restrained channel colors only in their established contexts.
- Use one coherent color family within a section.
- Keep colors pigmentary rather than luminous or synthetic.
- Keep color quiet enough for artwork to remain dominant.
- Preserve readable contrast and never rely on color alone for meaning.

## Never

- Introduce saturated corporate colors.
- Add colors without an organizational or semantic role.
- Use decorative gradients, neon, or glow.
- Apply unrelated colors to items that belong to the same group.
- Let section color overpower image content.
