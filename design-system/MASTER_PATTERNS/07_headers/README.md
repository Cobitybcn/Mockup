# Master Pattern: Headers

## Pattern

Headers establish an editorial entry into a workspace. A prominent serif title identifies the worktable while concise supporting information and restrained utilities orient the immediate task.

## Invariants

- Give the principal title an editorial serif character.
- Use discreet sans-serif typography for supporting information.
- Keep instructions short and practical.
- Preserve generous space around the heading group.
- Keep utilities, counts, and status subordinate.
- Maintain a clear relationship between the header and the workspace it introduces.

## Never

- Turn the header into a dashboard summary.
- Lead with metrics, filters, or administrative controls.
- Use long introductory paragraphs.
- Create several titles with equal visual weight.
- Replace the editorial character with generic framework typography.
