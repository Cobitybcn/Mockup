# Master Pattern: Workspace Panels

## Pattern

Workspace Panels are large, calm working surfaces containing one stage of visual production. They provide room to inspect, compare, move, and assign imagery without behaving like dashboard modules.

## Invariants

- Use a light surface and thin boundary.
- Preserve generous internal white space.
- Give images and active production material priority over controls.
- Keep the heading and immediate instruction concise.
- Fold secondary configuration when it is not required.
- Keep feedback within the panel that produced it.
- Maintain a clear place within the vertical workspace flow.

## Never

- Treat a panel as a dashboard widget.
- Fill it with metrics, KPI summaries, or administrative reports.
- Create several nested surfaces with equal emphasis.
- Use heavy shadow or decorative elevation.
- Use long prose or giant forms as its primary content.
