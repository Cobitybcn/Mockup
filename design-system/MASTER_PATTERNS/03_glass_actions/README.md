# Master Pattern: Glass Actions

## Pattern

Glass actions are small circular icon controls placed over images. Their restrained frosted appearance keeps actions available while allowing the image to remain dominant.

## Invariants

- Use a small circular form.
- Use a frosted or translucent surface with low visual weight.
- Keep every action on the image it affects.
- Place favorites at the top-left.
- Place secondary actions at the top-right.
- Use a clear icon without visible button text.
- Provide an accessible name and a concise tooltip where clarification is needed.
- Preserve legibility over varied image content.

## Never

- Create toolbars below thumbnails.
- Use exaggerated blur, glow, or glassmorphism.
- Cover important artwork content.
- Use misaligned typographic symbols instead of established icons.
- Duplicate the same image action elsewhere.
