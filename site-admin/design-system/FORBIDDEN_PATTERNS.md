# Artist Site Manager Forbidden Patterns

- No SaaS dashboard, KPI wall, analytics tiles, or generic admin template.
- No copy of the public website header or visitor navigation.
- No copy of the Artwork Mockups studio shell.
- No saturated commerce palette, decorative gradient, glow, or heavy shadow.
- No giant permanent forms; fold secondary settings.
- No duplicated edit controls across Artwork Mockups and Site Manager.
- No image entity represented only as a text row when an image is available.
- No publication on every keystroke.
- No editorial synchronization that changes price, stock, orders, payments, or shipping.
- No public-site CSS change as a side effect of a Site Manager feature.
- No destructive action presented as the primary action.
