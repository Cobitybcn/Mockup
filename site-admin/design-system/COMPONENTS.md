# Artist Site Manager Components

## Manager Header

Identifies the current artist site, contains primary navigation, and offers quiet links back to Artwork Mockups and to the public site without creating a second navigation row.

## Primary Tabs

Four text tabs — Content, Store, Settings, Activity — with a fine active underline. They are navigation, not pill buttons or dashboard cards.

## Secondary Tabs

Short text links for the active domain. Content preserves the public navigation order. On narrow screens they may scroll horizontally without shrinking text.

## Visual Rail

A horizontal collection of image-led cards for artworks, series, notes, and print groups. Movement is user-directed.

## Visual Card

A compact rectangular image-led selection surface. State is a small label and metadata stays on one short line where possible. Several cards must remain visible on a 15-inch laptop. Selecting the card opens its editor.

## Editor Panel

A broad, quiet workspace. The primary image and essential fields remain visible; optional configuration uses folded sections.

## Primary Action

One soft-pastel commitment per workspace, such as Publish, Update website, or Save print. Secondary and destructive actions remain visually subordinate.

## Status Label

Compact text for Draft, Published, Pending changes, Hidden, Low stock, or Sold out. Meaning never depends on color alone.

## Operational Table

Reserved for orders and audit records. It uses comfortable rows, fine separators, and no dashboard metrics.
